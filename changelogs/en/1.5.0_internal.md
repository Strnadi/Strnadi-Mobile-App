# Changelog

## 1.5.0 - 2026-02-26

Comparison range: `23651b8` (last commit with `version: 1.4.20+1.4.20`) -> `WORKING_TREE` (pending commit)

### Added
- New `Blog` tab in bottom navigation with `BlogExplorerContent`.
- Article category fetch support (`/articles/categories?articles=true`) and uncategorized article fallback list.
- In-app article search and pull-to-refresh in blog explorer.
- New `GuidePage` and `GuideShortcutButton` for fast guide access from recording.
- Reusable `NotificationBellButton` with unread badge backed by local notification state.
- Notification unread helpers in `DatabaseNew` (`getUnreadNotificationCount`, `refreshUnreadNotificationCount`, `markAllNotificationsAsRead`).

### Changed
- Refactored shared navigation scaffold into `lib/navigation/scaffold_with_bottom_bar.dart` and extracted guest-user popup helper.
- Notification screen now marks notifications as read on open and hides duplicate bell while in notifications.
- Markdown rendering now supports relative article links/images and improved visual readability.
- Google auth/sign-up calls now include email when available, with idToken-based fallback extraction.
- Map overlays and bottom sheets now account for bottom safe-area insets.
- Settings menu flow simplified; guide content moved to dedicated navigation.
- Localization updated (EN/CS/DE) for blog explorer and wording change from cache deletion to device deletion.

### Version
- `pubspec.yaml`: `1.4.21+1.4.21` -> `1.5.0+1.5.0`

### Test Focus
- Blog tab behavior: load, search, refresh, category expansion, article open.
- Guide shortcut on recorder and return navigation.
- Notification badge updates, mark-as-read behavior, and guest-user restrictions.
- Google sign-in/sign-up behavior when account email is absent but present in ID token.
- Markdown article link/image resolution and map control spacing on devices with bottom insets.
